Case Study: Autonomous Line-Following Drone

Project Overview

As part of the MathWorks Mini Drone Competition, our team, SCAL, developed an autonomous line-following drone using a Parrot Mini Drone. The system leverages real-time image processing and path planning algorithms implemented in MATLAB and Simulink to enable the drone to track a red line and land precisely on a designated circular target. This project showcases advanced control systems, computer vision, and embedded programming tailored for autonomous navigation.

Objectives

The primary objectives were to:

  • Develop a robust algorithm for detecting and following a red line using the drone’s onboard camera.
  • Implement precise landing capabilities on a circular target using vision-based detection.
  • Ensure dynamic path correction and obstacle avoidance for stable navigation.
  • Compete successfully in the MathWorks Mini Drone Competition by demonstrating reliable performance.

Challenges

  • Sharp Turns: The drone struggled with navigating turns sharper than 60 degrees, often overshooting the path.
  • Circle Detection for Landing: Premature or missed landings occurred due to inaccurate speed calculations and ROI detection.
  • Lighting Variability: Inconsistent lighting affected the accuracy of red line detection.
  • Real-Time Processing: Balancing computational efficiency with real-time performance on the drone’s limited hardware.

Solutions

To address these challenges, we implemented:

  • Line Detection: Utilized Canny edge detection and Hough Line Transform in MATLAB to identify the red line, followed by line selection algorithms to determine the most prominent line for heading calculation.
  • Bitmap Intersection Analysis: Developed a bitmap-based approach to refine the drone’s heading, incorporating X-mark generation for trajectory stabilization.
  • Circle Detection: Employed MATLAB’s `imfindcircles()` function to detect the landing circle, with dynamic ROI adjustments for precise landing decisions.
  • Path Planning: Integrated a 1st order Butterworth filter (0.3 Hz cutoff) to smooth noisy vision data and a MATLAB function-based logic for real-time path corrections and speed adjustments.
  • Obstacle Avoidance: Used black region detection to identify boundaries and avoid obstacles, ensuring safe navigation.

Technologies Used

  • Hardware: Parrot Mini Drone (with onboard camera)
  • Software: MATLAB, Simulink, Computer Vision Toolbox
  • Programming Languages: MATLAB, C++ (for performance-critical sections)
  • Algorithms: Canny Edge Detection, Hough Line Transform, Bitmap Intersection Analysis, `imfindcircles()`
  • Communication Interface: Wi-Fi
  • Other Tools: Butterworth Filter, MATLAB Function Blocks

Results

The drone successfully tracked a red line and landed on a circular target in controlled environments, demonstrating robust performance in the MathWorks Mini Drone Competition. Key achievements included:

  • Accurate line following with real-time path corrections.
  • Stable landing on designated targets with improved ROI detection.
  • Effective obstacle avoidance using black region analysis.
  • Enhanced team collaboration and problem-solving under competitive constraints.

Improvements Suggested

Based on the challenges faced, we proposed:

  • Path Prediction for Sharp Turns: Implement predictive algorithms to anticipate sharp turns and adjust speed dynamically.
  • Dynamic ROI for Landing: Develop a speed-dependent ROI calculation to improve landing accuracy.
  • Lighting Adaptation: Enhance image preprocessing to handle varying lighting conditions.

Conclusion

This project demonstrated the successful integration of image processing, control systems, and path planning in an autonomous drone. Participating in the MathWorks Mini Drone Competition provided invaluable experience in applying ECE concepts like computer vision and embedded systems in a real-world context. The project also highlighted the importance of teamwork, led by Naman Jain, and problem-solving under competitive conditions.
